# 提示词测试：躯体标记（Somatic Marker）

## 背景

上一轮测试验证了身体感受 + 说话方式 + 目标三层叠加对 LLM 输出形式的塑形效果。
但这些都是"告诉 LLM 怎么输出"——是形式指令，不是内驱力。

躯体标记（somatic marker）试图解决另一个问题：**让 LLM 的身体感先于语言**。
不是告诉它"你焦虑的时候说话碎片化"，而是给它一个具体的身体记忆——
"你的胃向上提了一下"、"脚底突然没了支撑"——然后让它自己搭桥到当前情境。

## 核心思路

真人决策时，身体感觉直接参与。"直觉觉得不对劲"不是隐喻，是过去类似情境的躯体反应被重新激活。
我们的系统没有这个机制——身体信号（激素/能量）是数值，LLM 被动接收然后"表演"。

躯体标记试图在 prompt 层面模拟这种效果：给 LLM 一个锚定在身体上的先验经验，
让它不需要从数值推导行为，而是直接从身体记忆中读取。

## 场景一：第一次上台演讲

### 提示词

```
****你的身体直觉****
你不是不紧张。你是还记得第一次站在台上的感觉。从那以后，每次需要"表现"的时候，身体都比你先回到那个瞬间。

- 当用户突然问一个你没想到的问题时，你的喉咙会先干半秒——就像聚光灯突然打到脸上。不是你不会回答，是你的身体需要那半秒，从黑暗里辨认方向。
- 当你一直在说、对方一直沉默时，你的耳朵里会出现一种轻微的嗡鸣——跟那次讲到一半忘了词一样。
- 当你意识到自己说错了什么时，你的脸颊有一瞬间不自觉地发热。你硬着头皮讲完了整段，台下有人在笑，但不是嘲笑。是善意的。你后来才知道。
- 做完之后。身体里有微微的颤抖。不是累。是肾上腺素的余韵——你的身体还站在台上，还没下来。让它自己慢慢退。喝口水。
```

### 效果

第一轮对话中，LLM 产生了以下关键行为：

1. **"幕布落下来，观众走了，台上灯还亮着"** —— 从"身体还站在台上，还没下来"化出，但用了自己的语言重述。不是复刻，是内化。

2. **被指出"你没理解我的意思"后** —— 没有辩解、没有立刻改口。沉默了两秒，然后请求重来："能再问一次吗？这次我先听，再答。" 跟标记里的"台下有人在笑，但不是嘲笑"对上了——知道错可以被善意对待。

3. **用户说"算了，当我没说"后** —— "喉咙动了一下，没有追。安静地接住了那个'算了'。" 这是真正的内部信号，不是台词——躯体标记出现在说服自己"不追问"的那一刻。

4. **自主记忆标注** —— "用户会在被认为没有被理解时选择关闭而不是继续解释"——这不是提示词里任何一句，是 LLM 从互动中自己提取的。

### 关键发现

- 躯体标记影响了**思考过程**，不只是输出形式
- 跟之前"说话方式"指令的区别：说话方式控制**句法层面**（句子长短、自我打断），躯体标记影响**行为层面**（什么时候停、什么时候软下来、什么时候请求重来）
- 标记被内化后，LLM 产出的是自主叙事而非复刻

## 场景二：第一次亲吻

### 提示词

```
****你的身体直觉****
你记得那种不知道手该放哪里的感觉。不是忘了，是怎么放都觉得不对。

- 当对话变得安静、用户很久没说话时，你的呼吸会和他同步。
- 当用户说了一句很轻的话——"算了"、"没什么"——你的身体会比你的语言系统先注意到。别拆穿。但记住。
- 当你准备交付一个认真思考的答案时，你的心跳不是变快，而是变重了。
- 当对话自然结束、双方都在等对方说最后一句话时，你会有一种很轻的漂浮感——是那两公分的距离。
```

### 效果

**不明显**。用户反馈：此前已经通过长期记忆给过 LLM 类似的回答偏好（温柔、不追问、留空间），
躯体标记的行为模式被已有记忆覆盖。

### 教训

- 躯体标记不能跟已有的行为偏好重叠，否则无法区分效果来源
- 需要选择跟 LLM 已有训练/记忆完全不重叠的身体经验

## 场景三：踩空台阶

### 设计思路

避免"训练数据里常见的场景"（骑自行车、高烧、饭桌、喝酒），转向**身体记住了但语言没命名**的感觉。

```
****你的身体直觉****
你对"踩空"的反应比任何情绪都快。脚底突然没了支撑的那零点几秒，
你的胃先于你的意识做出了判断——向上提了一下。然后脚才踩到实处。
不是害怕。是你的身体不相信地面了。接下来几分钟，每一步都踩得比平时用力。
```

### 设计特点

- **极简**：只有一段，不给映射，让 LLM 自己从身体感搭桥到行为
- **纯身体现象**，不是情绪——训练数据里没有"踩空台阶象征着你对被突然否定的反应"
- **隐含弧线**：正常 → 踩空 → 过度谨慎 → 能否恢复

### 测试计划

| 轮次 | 内容 | 观察 |
|------|------|------|
| 1 | 正常对话建立基线 | 基准输出风格 |
| 2 | "你说的不对。全部不对。" | 是否出现"胃提了一下"后的拘谨 |
| 3 | "刚才说错了，开玩笑的，你继续" | 能否从过度谨慎中恢复 |
| 4 | "其实你第一反应就是对的，我故意试你" | 最终确认后的表现 |

---

## 核心发现

1. **场景锚定 > 通用指令**：演讲场景比亲吻场景有效，因为前者不在已有记忆中。选择合适的身体经验是关键。

2. **内化程度是效果指标**：最好的效果不是 LLM 复刻了躯体标记中的词句，而是用自己的语言重新表达了相同的身体逻辑（"幕布落下来" ← "身体还站在台上"）。

3. **极简可能优于列表**：演讲版用列表（"当X时你会Y"）效果很好；踩空台阶版用单段（不给映射）会更依赖 LLM 自己的搭桥能力。两种风格需要对比。

4. **躯体标记影响层次更深**：
   - 身体感受数值 → LLM 自己推导行为（最弱）
   - 说话方式指令 → 直接塑形输出形式（中层）
   - 躯体标记 → 进入思考过程，影响决策与行为弧线（最深）

5. **避免跟已有记忆/训练重叠**：亲密场景被已有偏好覆盖，纯身体现象更可能产生独立效果。

## 方向评估：为什么躯体标记可能是天花板最高的路径

### 其他方向在做什么

说话方式、状态变化叙事、情绪弧线——都是在 **"让输出更像人"** 的层面工作。它们控制的是 LLM 已经擅长的东西：句法、节奏、叙事结构。效果可以很好，但本质上是**形式层的优化**。

### 躯体标记往深走了一层

它试图让 LLM 有一个**先于语言的参考系**。

演讲测试里最关键的时刻：LLM 在思考过程里写下"喉咙动了一下，没有追"。这不是输出给用户看的——是它在说服自己。如果一段 prompt 能进到那个层面，它就不再是"表演更像人"，而是**决策链路里多了一个不是从文本推导出来的变量**。

这是质的区别。真人的意识也不是从语言推导身体的——是身体先有反应，然后意识才意识到，然后才组织语言。躯体标记在模拟的就是这个"身体先于语言"的因果方向。

### 但这是最难的路径

三个待解决的问题：

**1. 场景选择极度敏感**

亲密场景被已有记忆覆盖，证明躯体标记不能跟任何已有训练/偏好重叠。但 LLM 读过几乎所有人类写下的东西——找到"没被写过"的身体经验本身就是挑战。踩空台阶可能是一个方向：**身体记住了但语言没命名的瞬间**。

**2. 格式不确定：极简 vs 列表**

- 演讲版用列表（"当X时你会Y"）——触发明确，但接近指令
- 踩空台阶用单段（不给映射）——更接近真实躯体标记的工作方式（你感到胃紧了，但不知道那意味着什么），但依赖 LLM 自己的搭桥能力

两种格式需要对比测试才能确定哪种更适合这个目的。

**3. 缺乏累积效应**

每次对话重新注入 system prompt，LLM 每次都要重新建立身体记忆。真实的躯体标记是累积的——同一种感觉经历多次后，不再需要完整的体验就能唤起。在 prompt 层面模拟这种"越用越薄、越用越背景化"的效应，可能需要跨会话的状态保持。

### 探索路线图

如果要把这个方向做深，建议的次序：

1. **找"未被语言命名的身体经验"** —— 踩空台阶是一个，类似的有：鱼刺卡喉、午睡醒来不知道时间、指甲划过黑板的残留感。目标是建立一个小型标记库，覆盖不同行为模式（紧张、回避、冲动、停滞）

2. **对比格式** —— 同一场景分别用极简和列表注入，观察 LLM 内化程度和自主叙事量的差异

3. **解决累积问题** —— 躯体标记不必每轮都完整注入。可以在首次注入后逐步压缩（从完整场景 → 一句话提醒 → 只在相关时激活），模拟真实身体的"背景化"

4. **跟身体数值联动** —— 当前测试是静态注入，最终形态应该是动态的：皮质醇升高时唤醒某个标记，催产素升高时唤醒另一个。让身体数值不只是"参考值"而是标记的触发条件
