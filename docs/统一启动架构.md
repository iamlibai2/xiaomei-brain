# xiaomei-brain 统一启动架构

## 一、启动流程图

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         main()                                     │
│  1. Config.from_json()        # 加载配置                           │
│  2. AgentManager(base_dir)     # 初始化 Agent 管理器                 │
│  3. 加载 bindings              # 获取 channel→agent 路由规则          │
│  4. AgentGateway(...)          # 创建 Gateway（带路由逻辑）            │
│  5. load_channels_from_config() # 从配置加载所有 Channel              │
│  6. 预加载 embedding 模型       # 初始化向量检索                    │
│  7. uvicorn.run()             # 启动 WebSocket 服务器               │
└─────────────────────────────────────────────────────────────────────────┘
```

## 二、详细启动步骤

```python
# Step 1: 加载配置
config = Config.from_json()
# 读取 ~/.xiaomei-brain/config.json
# 解析：providers、agents、channels、bindings、xiaomei_brain section

# Step 2: 初始化 AgentManager
agent_manager = AgentManager(base_dir="~/.xiaomei-brain")
# 扫描 ~/.xiaomei-brain/agents/{id}/talent.md
# 创建 AgentInstance 注册表（lazy，不加载 LLM/Memory）

# Step 3: 创建 Gateway（带路由）
gateway = AgentGateway(
    agent_manager=agent_manager,
    config=config,
    bindings=[               # 路由规则
        {"agentId": "xiaomei", "match": {"channel": "feishu", "accountId": "xiaomei"}},
        {"agentId": "xiaoming", "match": {"channel": "feishu", "accountId": "default"}}
    ]
)

# Step 4: 加载 Channels
for account_id, account_cfg in config["channels"]["feishu"]["accounts"].items():
    channel = FeishuChannel(
        app_id=account_cfg["appId"],
        app_secret=account_cfg["appSecret"],
        account_id=account_id,  # 用于路由
    )
    gateway.add_channel(channel)

# Step 5: 启动所有 Channels
await gateway.start_all()
# 对每个 channel 调用 channel.start(gateway.handle_inbound)
# FeishuChannel 启动独立的 WebSocket 线程接收消息

# Step 6: 启动 WebSocket Server（Uvicorn）
uvicorn.run(app, host="0.0.0.0", port=8765)
```

## 三、消息处理流程

```
用户发送消息 (飞书)
      ↓
┌─────────────────────────────────────────────────────────────┐
│ FeishuChannel._on_message() [WebSocket 线程]             │
│  1. 解析 lark-oapi 事件                                  │
│  2. 创建 InboundMsg(platform="feishu", account_id="...")  │
│  3. 放入 queue.Queue()                                   │
└─────────────────────────────────────────────────────────────┘
      ↓
┌─────────────────────────────────────────────────────────────┐
│ FeishuChannel._process_messages() [asyncio 协程]        │
│  1. 从 queue.Queue() 取出消息                             │
│  2. 调用 gateway.handle_inbound(msg)                    │
└─────────────────────────────────────────────────────────────┘
      ↓
┌─────────────────────────────────────────────────────────────┐
│ AgentGateway.handle_inbound()                            │
│  记录日志 → 调用 self.on_message(msg)                  │
└─────────────────────────────────────────────────────────────┘
      ↓
┌─────────────────────────────────────────────────────────────┐
│ AgentGateway.on_message() [路由逻辑]                     │
│  1. 检查 msg.extra.get('agent_id')                      │
│  2. 否则查找 bindings: channel + account_id → agent_id   │
│  3. 调用 agent_manager.build_agent(agent_id, config)     │
│  4. agent.stream(msg.text) 生成回复                     │
└─────────────────────────────────────────────────────────────┘
      ↓
┌─────────────────────────────────────────────────────────────┐
│ FeishuChannel.send() [async]                             │
│  使用 lark-oapi HTTP API 发送回复到飞书                  │
└─────────────────────────────────────────────────────────────┘
```

## 四、核心组件架构

```
start_all.py (主进程)
│
├── AgentManager
│   └── _agents: dict[str, AgentInstance]
│       ├── "xiaomei" → AgentInstance(llm, tools, memory...)
│       └── "xiaoming" → AgentInstance(llm, tools, memory...)
│
├── AgentGateway (继承 Gateway)
│   ├── channels: List[Channel] = [FeishuChannel, FeishuChannel]
│   ├── _bindings: List[Dict]      # 路由规则
│   └── _agents: Dict[str, Agent]  # 缓存的完整 Agent 对象
│
├── FeishuChannel (实例1: account="xiaomei")
│   ├── WebSocket 线程: 接收飞书事件
│   ├── Queue: 跨线程消息传递
│   └── _process_messages 协程: 处理消息队列
│
├── FeishuChannel (实例2: account="default")
│   └── 同上...
│
└── Uvicorn WebSocket Server
    ├── /ws endpoint: Web 客户端连接
    └── /health endpoint: 健康检查
```

## 五、关键设计原理

#### 1. 线程/协程隔离

```
主事件循环 (asyncio.run(main()))
│
├── _process_messages 协程 (async)
│   └── 从 queue.Queue() 读取 → 处理消息
│
└── WebSocket 线程 (threading.Thread, daemon=True)
    └── lark-oapi SDK 专用事件循环
```

**原理**:
- `lark-oapi` SDK 在独立线程中运行，避免与主事件循环冲突
- `queue.Queue()` 实现线程间消息传递（线程安全）
- 协程处理业务逻辑，线程处理 IO

#### 2. 路由机制

```python
# 路由查找顺序：
def _get_agent_for_msg(msg):
    # 1. 显式指定 (WebSocket)
    if msg.extra.get("agent_id"):
        return get_agent(msg.extra["agent_id"])

    # 2. Bindings 路由 (Channel)
    account_id = msg.extra.get("account_id", "default")
    for binding in self._bindings:
        if binding["match"]["channel"] == msg.platform and \
           binding["match"]["accountId"] == account_id:
            return get_agent(binding["agentId"])

    # 3. 默认
    return get_agent("xiaomei")
```

**配置示例**:
```json
{
  "channels": {
    "feishu": {
      "accounts": {
        "xiaomei": {"appId": "cli_xxx"},  → 绑定到 xiaomei
        "default": {"appId": "cli_yyy"}     → 绑定到 xiaoming
      }
    }
  },
  "bindings": [
    {"agentId": "xiaomei", "match": {"channel": "feishu", "accountId": "xiaomei"}},
    {"agentId": "xiaoming", "match": {"channel": "feishu", "accountId": "default"}}
  ]
}
```

#### 3. Lazy Loading

```python
class AgentManager:
    def build_agent(self, agent_id, config):
        instance = self._agents.get(agent_id)  # 仅元数据
        if instance.llm is None:              # 首次访问才加载
            instance.llm = LLMClient(...)
            instance.memory = MemoryStore(...)
            instance.tools = ToolRegistry(...)
        return instance
```

**优点**:
- 启动快：不预先加载所有 LLM/Memory
- 按需：只有用到的 agent 才初始化
- 节省：不用的 agent 不占资源

#### 4. Agent 构建流程

```
AgentInstance (元数据)
    │
    ├── llm: None → LLMClient(model, api_key)
    ├── tools: None → ToolRegistry() + 注册工具
    ├── memory: None → MemoryStore(embedder, index)
    ├── session_manager: None → SessionManager(session_dir)
    └── working_memory: None → WorkingMemory()
    │
    ↓
Agent (完整执行引擎)
    ├── llm: LLMClient
    ├── tools: ToolRegistry (包含 shell, file_ops, memory...)
    ├── memory: MemoryStore (向量检索)
    ├── dream_scheduler: DreamScheduler (定期处理记忆)
    ├── proactive_engine: ProactiveEngine (主动消息)
    ├── reminder_manager: ReminderManager (提醒管理)
    └── context_extractor: ContextExtractor (上下文提取)
```

## 八、关键代码解释

```python
# AgentGateway: 核心路由逻辑
class AgentGateway(Gateway):
    def __init__(self, agent_manager, config, bindings):
        self.agent_manager = agent_manager  # Agent 工厂
        self.config = config              # 全局配置
        self._bindings = bindings         # 路由规则
        self._agents = {}               # Agent 缓存

    def _get_agent_for_msg(self, msg):
        # 懒加载 + 缓存
        agent_id = self._resolve_agent_id(msg)
        if agent_id not in self._agents:
            instance = self.agent_manager.build_agent(agent_id, self.config)
            self._agents[agent_id] = self._build_full_agent(instance)
        return self._agents[agent_id]
```

```python
"""WebSocket 客户端测试脚本 - 与 xiaomei-brain 对话"""

import asyncio
import json
import sys

async def chat(host: str = "127.0.0.1", port: int = 8765, agent_id: str = "xiaomei"):
    """Connect to WebSocket and chat."""
    import websockets

    uri = f"ws://{host}:{port}/ws"
    print(f"Connecting to {uri}...")
    print(f"Agent: {agent_id}")
    print("-" * 50)

    async with websockets.connect(uri) as ws:
        # Start session
        await ws.send(json.dumps({
            "type": "session_start",
            "session_id": "test-session",
            "agent_id": agent_id,
        }))
        response = await ws.recv()
        print(f"[Server] {response[:100]}...")

        print("-" * 50)
        print("Connected! Type your message (or 'quit' to exit):")
        print("-" * 50)

        # Chat loop
        while True:
            # Read user input
            user_input = input("\nYou: ").strip()
            if not user_input:
                continue
            if user_input.lower() in ("quit", "exit"):
                break

            # Send message
            await ws.send(json.dumps({
                "type": "chat",
                "content": user_input,
                "session_id": "test-session",
            }))

            # Receive response (streaming)
            full_response = ""
            while True:
                msg = json.loads(await ws.recv())
                msg_type = msg.get("type")

                if msg_type == "text_chunk":
                    # Stream chunk
                    chunk = msg.get("content", "")
                    print(chunk, end="", flush=True)
                    sys.stdout.flush()
                    full_response += chunk
                elif msg_type == "text_done":
                    print()  # New line
                    break
                elif msg_type == "error":
                    print(f"\n[Error] {msg.get('message')}")
                    break

        # End session
        await ws.send(json.dumps({
            "type": "session_end",
            "session_id": "test-session",
        }))
```
