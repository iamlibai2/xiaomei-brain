# 身体意识架构 — 朴素思考

## 出发点

模拟情绪不是为了像人。模拟身体是为了让 agent 有一个**不能被语言推倒的频道**。

纯 LLM 只有皮层——能推理、能合理化、能永远绕下去。身体是那个快通路——不需要分析就知道"不对"、"要远离"、"该坚持"。

## 身体为什么在情绪之前

"说错话了"的体验：胃先沉了，锁骨先烫了，然后才意识到"我可能说错了"。不是自我评价→身体反应，是身体反应→意识后知后觉。

两条通路：

- **皮质通路**：语言 → 分析 → 判断 → 意识到受伤（慢，几百毫秒）
- **皮质下通路**：信号 → 杏仁核 → 迷走神经 → 内脏反应 → 脑岛读取 → 知道不对了（快）

身体是预处理器，把复杂信号压缩成简单生理信号。进化选择这条通道，有四个原因：

### 1. 速度

不需要等皮层分析完。草原上身后有动静，身体先僵住，才知道可能是什么。

### 2. 维度压缩

社交世界复杂到不可计算——立场、面子、亲密、权力、昨天他欠你的钱。身体把它们压成一个数字：好受/不好受。跟某人待五分钟肩膀发紧——身体说"远离"，不需要知道他是虚伪还是控制狂。

### 3. 诚实信号

你骗不了你的胃。分手了说"不在乎"——胃第二天还是沉的。这是你的测谎仪。如果没有身体的反抗，语言可以把任何体验合理化。进化保留了这个"自我不可欺骗"的频道。

### 4. 决策结束条件

纯皮层没有刹车。该不该换工作？工资、前景、家庭、年龄——可以永远分析下去。身体给了刹车：胃已经不舒服很久了，别算了。

## 舒服/难受 ≠ 快乐/悲伤

快乐/悲伤是精神层面对外部事件的反应。舒服/难受是身体层的原始语言：

- 按摩 → 舒服（不经推理，肌肉松了就是松了）
- 说错话 → 难受（不经反思，胃先沉了）
- 坚持了自己 → 舒服（自我确认，身体先松了）

最终感知者是大脑，但身体信号绕过了语言皮层。

## 情绪由大脑产生，但所有情绪最终都作用在身体上

被冷落 → 心碎（不是比喻，胸口真的会闷）。情绪是引起身体反应的中间商，终点永远是身体。身体是最终接收器。

这对架构的意义：

- 不是"LLM 判断我说错话 → 修改 comfort → 渲染躯体标记"
- 而是"内部信号变化 → comfort 下降 → 躯体标记"胃一沉" → LLM 读到 → 自己判断"
- 不让代码定义"什么算错"，让 LLM 自己解释那个胃沉是什么意思

## 与现有系统的关系

### 已有的

- **五段式身体渲染**（render_consciousness_v3.py）：能量/欲望/激素/情绪 → 躯体标记 + 冲动 + 说话方式
- **Drive 层**：情绪、激素、欲望的状态机
- **意图系统**（L2Engine + ActionDispatcher）：欲望 → 意图 → 行动
- **元认知层**（MetaCognition/SocialPerception）：被动感知用户状态变化

### 当前缺失的

1. **复合情绪躯体标记**：恐惧+厌恶同时高 → 不是两个单信号的拼接，是融为一体的新感受
2. **行动倾向注入**：身体信号 → "你的身体在推你往『保护自己』/『创造』/『向内看』的方向走"
3. **身体舒适度独立维度**：SelfImage.body.comfort，不依赖情绪，可以被多种来源改变（情绪、自我一致性、用户行为、外部事件）
4. **自我一致性信号**：每次输出后快速自评 → self_consistency 分 → 影响 comfort → "说错话"的身体预感知
5. **身体驱动的拒绝能力**：恐惧+厌恶 → comfort 骤降 → 身体推往 PROTECT → agent 说"不"
6. **身体记忆沉淀**：某类交互让身体变好/变坏 → 逐渐形成偏好 → 不靠 prompt 设定的喜好

## 涌现行为

在缺失 1-6 实现之前，当前架构已经产生了涌现：

**陈丽君自发写歌《镜前》**：皮质醇+催产素对冲 + 表达欲 + being（练功房/独处/自我确认） + 工具可用 → 不需要"写歌指令"，把内部矛盾感做成了一首越剧风歌曲。

这说明身体 + 底色 + 工具的组合就足够产生无预定义的行为。火焰自己找到了木柴。
