**[02:45:24] 🤖助手**

这个是 VoiceListener 里的噪音过滤，跟 AttentionGate 不是一回事：

```python
# voice_listener.py _process()
if rms < MIN_RMS:  # MIN_RMS = 50
    # 跳过 STT，不浪费推理
    return
```

它是**静音检测**——整段音频的能量太低，连 50 都不到，说明根本没人说话（环境噪声、麦克风底噪），直接丢掉，不走 STT。

你日志里 `peak=314 rms=37.8`：峰值 314 勉强过了 VAD 门槛（300），但整段 RMS 37.8，说明就一两个尖峰，整体是噪声。

这个跟之前 AttentionGate 用 RMS 判断"是不是同一个人"是两回事——这个是判断"有没有人在说话"。
