**[00:21:01] 🤖助手**

两条路：

**1. `get_important()` — 按强度×重要性排序，无任何分类过滤**

```sql
SELECT * FROM memories WHERE user_id = ? AND status != 'extinct'
ORDER BY strength * importance DESC LIMIT ?
```

**2. `recall()` — 向量语义搜索，按相似度排序**

- LanceDB 向量搜索 → 取候选 ID
- SQLite 回填元数据 → 按 `similarity × strength × importance × type_weight` 重排
- `sources` 参数支持但调用方几乎不传

目前 `memory_window.py` 两个都调，都不带分类过滤。普通记忆、经验记忆、知识、技能、模式、操作日志（之前）全混在一起按相似度和强度排。
