## 意识系统 ActionDispatcher 链路修复

### 问题：Intent 执行后未消费

**现象**：GREET 意图匹配后发送了主动消息，但 `intent_buffer` 中的 `"GREET"` 没有被清除，导致下一 tick 又被匹配，形成死循环。

**根因**：`_do_proactive()` 和 `_do_trigger_l3()` 执行完动作后，没有清除 `SelfImage.intent_buffer` 中对应的 intent。

**修复**：在 `action_dispatcher.py` 的 `_do_proactive()` 和 `_do_trigger_l3()` 中，执行完后从 `SelfImage.intent_buffer` 清除对应的 intent：

```python
# _do_proactive 中
if intent_type and item.source == "intent":
    si = self.dispatcher._get_self_image()
    if si and hasattr(si, "intent_buffer"):
        upper_type = intent_type.upper()
        si.intent_buffer = [i for i in si.intent_buffer if i.upper() != upper_type]
```

同理适用于 `_do_trigger_l3()`。
