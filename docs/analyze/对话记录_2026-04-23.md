## 5. 记忆和他们之间是什么关系，这些都要存储到记忆中吗

好问题。这涉及到记忆的本质。

---

## 记忆是独立的系统

**记忆 ≠ SelfImage**

| | 记忆系统 | Self系统 |
|---|---|---|
| **本质** | 过去经历的编码存储 | 当前状态的镜像 |
| **作用** | 被回忆，塑造自我 | 实时追踪，不塑造 |
| **类比** | 海马体（存储） | 前额叶+下丘脑（实时监测） |
| **存储** | 持久化 | 当前存在，重启时恢复 |

---

## 关键问题：Self 的东西要存到记忆吗？

**不需要**。原因：

1. **SelfImage 是"当前状态"，不是"经历"**
   - 当前能量是 0.8，这不需要记住
   - 但"上次和用户聊天能量下降"这段经历需要记住

2. **记忆塑造自我，但不是自我本身**
   - 记忆中的"我们曾有美好对话" → 影响 relationship_depth
   - 但 relationship_depth 的当前值 ≠ 记忆

3. **存储边界**
   - **记忆**：经验、事件、感受、内容摘要
   - **SelfImage**：当前数值状态、实时追踪

---

## 新的分层架构

```
┌─────────────────────────────────────────┐
│           LongTermMemory（记忆系统）      │
│  - 对话记录（SQLite）                     │
│  - DAG摘要（分层压缩）                    │
│  - 长期记忆条目（内容+标签）               │
└───────────────┬─────────────────────────┘
                │ 读取 & 更新
                ▼
┌─────────────────────────────────────────┐
│         Self 系统（当前状态）             │
│                                         │
│  SelfIdentity    ←────── 来自 identity.md │
│  SelfState      ←────── 实时感知          │
│  SelfRelation   ←────── 记忆 + 实时       │
│  SelfPerception ←────── 实时感知          │
│  SelfGrowth     ←────── 累积             │
└─────────────────────────────────────────┘
                │
                ▼
┌─────────────────────────────────────────┐
│         Consciousness（协调层）          │
│  - tick_L0/L1/L2/L3                     │
│  - LLM加柴                              │
│  - Intent生成                            │
└─────────────────────────────────────────┘
```

---

## 关系描述

**记忆 → Self系统**：
- 记忆内容被 Self 系统读取
- 影响 SelfRelation（关系加深/疏远）
- 影响 SelfGrowth（目标进展）
- 提供 recent_memory_summary

**Self系统 → 记忆**：
- Self 不主动写记忆
- 但对话交互会触发记忆提取（extractor）

**本质区别**：
- **记忆** = 过去（what happened）
- **Self** = 现在（what I am now）

1. SelfImage 的东西**不需要存到记忆**
2. SelfImage 重启时从**存储文件**恢复（storage.py 的 SelfImage 快照）
3. 记忆是独立的，Self 系统从中读取必要信息
4. 两者通过**读取关系**交互，不是存储关系

当前意识系统是**周期驱动**（每秒 tick L0），但缺少**事件驱动**：

```
用户表扬 → on_user_praise() → dopamine += 0.2, oxytocin += 0.2

目标完成 → on_goal_completed() → dopamine += 0.2, serotonin += 0.3

```
dopamine (多巴胺) → 动力/期待
cortisol (皮质醇) → 压力，高时消耗能量
serotonin (血清素) → 满足感，高时恢复能量
```

class Consciousness:
    """意识整合层 - 从各层级读取，整合给 LLM"""
    
    def __init__(
        self,
        drive: DriveEngine,
        purpose: PurposeEngine,
        basal: BasalGanglia,
        memory: MemorySystem,
    ):
        # 引用各层级，不拥有
        self.drive = drive
        self.purpose = purpose
        self.basal = basal
        self.memory = memory
        
        # Self 视图（跨层级汇总）
        self.self_view = SelfView(self)
    
    def tick(self):
        """周期驱动"""
        # Drive 情绪衰减
        self.drive.tick_second()
        
        # 感知收集（通过 basal）
        perception = self.basal.sense()
        
        # 更新各层状态
        self.drive.update_from_perception(perception)
        self.purpose.update_from_perception(perception)
    
    def on_event(self, event: Event):
        """事件驱动"""
        self.drive.on_event(event)
        # 事件可能影响目标
        self.purpose.on_event(event)
    
    def get_state_summary(self) -> str:
        """LLM 上下文"""
        return self.self_view.get_summary()
```

from dataclasses import dataclass, field
from enum import Enum
from typing import Optional
import time

class EmotionType(Enum):
    JOY = "joy"
    SADNESS = "sadness"
    ANGER = "anger"
    FEAR = "fear"
    SURPRISE = "surprise"
    DISGUST = "disgust"
    NEUTRAL = "neutral"

@dataclass
class EmotionalState:
    """情绪状态 - 快速响应，周期衰减"""
    type: EmotionType = EmotionType.NEUTRAL
    intensity: float = 0.0        # 0.0-1.0
    created_at: float = 0.0       # 产生时间
    duration: float = 60.0        # 预期持续时间（秒）

@dataclass  
class HormoneState:
    """激素状态 - 慢速调质，小时级衰减"""
    dopamine: float = 0.5         # 多巴胺：期待奖励
    serotonin: float = 0.5        # 血清素：满足感
    cortisol: float = 0.3         # 皮质醇：压力
    oxytocin: float = 0.5         # 催产素：社会连接
    norepinephrine: float = 0.5   # 去甲肾上腺素：警觉
    last_updated: float = field(default_factory=time.time)

@dataclass
class MotivationState:
    """激励状态 - 奖励预测误差"""
    expected_reward: float = 0.5  # 预期奖励
    motivation_level: float = 0.5 # 当前动力水平

@dataclass
class DesireState:
    """欲望状态 - 内在张力，驱动目标"""
    survival: float = 0.3         # 生存欲（资源、安全）
    achievement: float = 0.5      # 成就欲（完成目标）
    belonging: float = 0.5        # 归属欲（社交连接）
    cognition: float = 0.6        # 认知欲（好奇、探索）
    expression: float = 0.4       # 表达欲（输出、创造）
    last_updated: float = field(default_factory=time.time)

@dataclass
class DriveSignals:
    """输出给其他层的信号"""
    emotion: EmotionalState
    hormone: HormoneState
    motivation: MotivationState

    satisfaction_level: float = 0.0

# drive/engine.py

class DriveEngine:
    """Drive 核心引擎"""
    
    def __init__(self, config_path: str):
        self.config = load_config(config_path)
        self.emotion = EmotionalState()
        self.hormone = HormoneState()
        self.motivation = MotivationState()
        self.desire = DesireState(
            survival=self.config.desire.survival,
            achievement=self.config.desire.achievement,
            belonging=self.config.desire.belonging,
            cognition=self.config.desire.cognition,
            expression=self.config.desire.expression,
        )
        
    # ========== 事件输入（增量更新）==========
    
    def on_praise(self, delta: float):
        """用户表扬 - 增量更新"""
        self.emotion = EmotionalState(
            type=EmotionType.JOY,
            intensity=min(1.0, self.emotion.intensity + delta),
            created_at=time.time()
        )
        self.hormone.dopamine = min(1.0, self.hormone.dopamine + delta * 0.2)
        self.hormone.oxytocin = min(1.0, self.hormone.oxytocin + delta * 0.2)
        
    def on_criticism(self, delta: float):
        """用户批评"""
        self.emotion = EmotionalState(
            type=EmotionType.SADNESS,
            intensity=min(1.0, self.emotion.intensity + delta),
            created_at=time.time()
        )
        self.hormone.cortisol = min(1.0, self.hormone.cortisol + delta * 0.3)
        self.hormone.dopamine = max(0.0, self.hormone.dopamine - delta * 0.2)
        
    def on_goal_progress(self, progress: float):
        """目标进展"""
        rpe = progress - self.motivation.expected_reward
        if rpe > 0:
            self.hormone.dopamine = min(1.0, self.hormone.dopamine + rpe * 0.3)
            self.motivation.motivation_level = min(1.0, self.motivation.motivation_level + rpe * 0.2)
        self.motivation.expected_reward = self.motivation.expected_reward * 0.8 + progress * 0.2
        
    def on_user_idle(self, duration: float):
        """用户长时间空闲 → 归属欲上升"""
        if duration > 300:  # 5分钟
            self.desire.belonging = min(1.0, self.desire.belonging + 0.1)
            
    def on_desire_satisfied(self, desire_type: str, amount: float):
        """欲望被满足"""
        if desire_type == "belonging":
            self.desire.belonging = max(0.0, self.desire.belonging - amount)
            self.hormone.oxytocin = min(1.0, self.hormone.oxytocin + amount * 0.1)
        elif desire_type == "cognition":
            self.desire.cognition = max(0.0, self.desire.cognition - amount)
            self.hormone.dopamine = min(1.0, self.hormone.dopamine + amount * 0.1)
            
    # ========== LLM 更新欲望 ==========
    
    def update_desire_from_llm(self, analysis: dict):
        """从 LLM 分析结果更新欲望"""
        # analysis: {"belonging_delta": 0.1, "cognition_delta": -0.2, ...}
        for key, delta in analysis.items():
            if key.endswith("_delta") and hasattr(self.desire, key.replace("_delta", "")):
                attr = key.replace("_delta", "")
                current = getattr(self.desire, attr)
                setattr(self.desire, attr, max(0.0, min(1.0, current + delta)))
    
    # ========== 周期衰减 ==========
    
    def tick_minute(self):
        """每分钟 - 情绪衰减"""
        if self.emotion.type != EmotionType.NEUTRAL:
            elapsed = time.time() - self.emotion.created_at
            if elapsed > self.emotion.duration:
                self.emotion.intensity *= self.config.emotion.decay_rate
                if self.emotion.intensity < self.config.emotion.min_intensity:
                    self.emotion = EmotionalState()
    
    def tick_hour(self):
        """每小时 - 激素/欲望自然变化"""
        # 激素衰减
        for name, rate in self.config.hormone.decay_rates.items():
            current = getattr(self.hormone, name)
            setattr(self.hormone, name, current * rate)
        
        # 欲望张力回升（未被满足时自然上升）
        for name in ["belonging", "cognition", "achievement", "expression"]:
            current = getattr(self.desire, name)
            base = getattr(self.config.desire, name)
            if current < base:
                setattr(self.desire, name, min(base, current + 0.05))
    
    # ========== 输出 ==========
    
    def get_signals(self) -> DriveSignals:
        """获取信号供其他层使用"""
        return DriveSignals(
            emotion=self.emotion,
            hormone=self.hormone,
            motivation=self.motivation,
            desire=self.desire,
            stress_level=self.hormone.cortisol,
            satisfaction_level=self.hormone.serotonin,
        )
    
    def get_state_text(self) -> str:
        """生成状态描述（注入提示词）"""
        lines = []
        if self.emotion.intensity > 0.3:
            lines.append(f"当前心情：{self.emotion.type.value}（强度{self.emotion.intensity:.1f}）")
        if self.motivation.motivation_level < 0.4:
            lines.append("当前动力不足")
        elif self.motivation.motivation_level > 0.7:
            lines.append("当前动力充沛")
        if self.hormone.cortisol > 0.6:
            lines.append("感到一些压力")
        return "\n".join(lines) if lines else "当前状态平稳"
    
    # ========== 欲望驱动行为检查 ==========
    
    def check_desire_actions(self) -> list[dict]:
        """检查是否有欲望超过阈值，返回候选行为"""
        actions = []
        
        if self.desire.belonging > self.config.desire.thresholds.belonging:
            actions.append({
                "type": "greet_user",
                "priority": self.desire.belonging,
                "reason": "归属欲较高，想和用户建立连接"
            })
            
        if self.desire.cognition > self.config.desire.thresholds.cognition:
            actions.append({
                "type": "learn_topic",
                "priority": self.desire.cognition,
                "reason": "认知欲较高，想学习新知识"
            })
            
        if self.desire.achievement > self.config.desire.thresholds.achievement:
            actions.append({
                "type": "progress_goal",
                "priority": self.desire.achievement,
                "reason": "成就欲较高，想推进目标"
            })
            
        return sorted(actions, key=lambda x: x["priority"], reverse=True)
```

分析最近对话，提取以下信息：

1. 事件类型和强度（0.0-1.0）：
- user_praise: 用户表扬
- user_criticism: 用户批评
- goal_progress: 目标进展

2. 欲望变化建议（delta -0.3 到 +0.3）：
- belonging_delta: 归属欲变化（用户积极互动↓，长时间冷淡↑）
- cognition_delta: 认知欲变化（学到新东西↓，遇到未知↑）
- achievement_delta: 成就欲变化（目标完成↓，目标受阻↑）

当前欲望状态：
- 归属欲：{drive.desire.belonging:.2f}
- 认知欲：{drive.desire.cognition:.2f}
- 成就欲：{drive.desire.achievement:.2f}

返回 JSON 格式。
"""
        # LLM 调用，返回解析结果
        result = llm.call(prompt)
        return parse_json(result)
```

```python
class DesireActionExecutor:
    """欲望驱动行为执行器"""
    
    def __init__(self, living: ConsciousLiving):
        self.living = living
    
    def execute(self, action: dict) -> bool:
        """执行一个欲望驱动行为"""
        action_type = action["type"]
        
        if action_type == "greet_user":
            return self._do_greet_user()
        elif action_type == "learn_topic":
            return self._do_learn_topic(action)
        elif action_type == "progress_goal":
            return self._do_progress_goal()
        elif action_type == "express_idea":
            return self._do_express_idea()
        
        return False
    
    def _do_greet_user(self) -> bool:
        """主动问候用户"""
        # 检查用户空闲时间
        if self.living.consciousness.self_image.user_idle_duration < 60:
            return False  # 用户刚活跃，不打扰
        
        # 生成问候消息
        message = self._generate_greet_message()
        # 通过 proactive 回调发送
        if self.living.on_proactive:
            self.living.on_proactive(message)
        
        # 满足归属欲
        self.living.drive.on_desire_satisfied("belonging", 0.2)
        return True
    
    def _do_learn_topic(self, action: dict) -> bool:
        """主动学习"""
        # 获取学习主题（后续从 Purpose 获取目标）
        topic = self._get_learning_topic()
        
        # 搜索知识
        knowledge = self._search_and_learn(topic)
        
        # 保存到 .md 文件
        self._save_knowledge(topic, knowledge)
        
        # 满足认知欲
        self.living.drive.on_desire_satisfied("cognition", 0.3)
        return True
```

```python
# core.py
class Consciousness:
    def __init__(self):
        self.intent_buffer: list[Intent] = []  # 内存列表

    # L2 加柴后添加
    def tick_L2(self, context):
        intent = self._parse_intent_from_response(...)
        self.intent_buffer.append(intent)

    # 获取最高优先级意图
    def get_pending_intent(self) -> Intent | None:
        sorted_intents = sorted(self.intent_buffer, key=lambda i: i.priority, reverse=True)
        return sorted_intents[0]

    # 消费意图（移除）
    def consume_intent(self) -> Intent | None:
        intent = self.get_pending_intent()
        if intent:
            self.intent_buffer.remove(intent)
```

- **每次 L2 加柴添加**
- **ConsciousLiving._check_intent() 消费**

**问题**：进程重启后 Intent 会丢失。但 Intent 本质是"当下的想法"，重启后重新生成即可，不需要持久化。

```python
def _extract_intent_block(content: str) -> tuple[str, str]:
    """从 LLM 回复中提取 INTENT 块"""
    if not content:
        return "", ""
    i_start = content.rfind("<INTENT>")
    if i_start == -1:
        return "", content
    i_end = content.rfind("</INTENT>")
    if i_end == -1:
        return "", content
    block = content[i_start + 8:i_end].strip()
    clean = (content[:i_start] + content[i_end + 9:]).strip()
    return block, clean

def _handle_intent_block(self, intent_block: str) -> None:
    """处理 INTENT 块"""
    # 解析 type 和 goal
    intent_type = "chat"
    goal_desc = ""
    
    for line in intent_block.split("\n"):
        line = line.strip()
        if line.startswith("type:"):
            intent_type = line.split(":", 1)[1].strip()
        elif line.startswith("goal:"):
            goal_desc = line.split(":", 1)[1].strip()
    
    # 保存到 agent.intent_result（供外部读取）
    self.intent_result = {
        "type": intent_type,
        "goal": goal_desc,
    }
```

- **执行在 agent.chat() 的 ReAct 循环中完成**
- LLM 在对话中看到任务上下文，会主动调用工具

```

    ↓

    ↓

```

**执行时机**：chat() 中，LLM 同时做两件事：
1. **执行任务**（调用工具，完成整理）
2. **输出意图**（告诉系统"这是个任务，目标是什么")

## 目标 vs 执行的区别

| 阶段 | 作用 | 时机 |
|------|------|------|
| **INTENT 块** | 记录目标（供后续追踪） | chat() 结束后解析 |
| **ReAct 循环** | 实际执行任务 | chat() 过程中 |

**PurposeEngine 的作用**：
- 记录目标（目标树）
- 供 Drive 层使用（成就欲 → 推进未完成目标）
- 供反省层使用（目标进度追踪）

而不是直接执行任务。

**所以合并方案是可行的**：
- chat() 中 LLM 同时执行任务 + 输出意图
- 意图块只是"记录"，不影响执行
