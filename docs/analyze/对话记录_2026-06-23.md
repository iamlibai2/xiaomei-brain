这是 **L2 意识涌现（`_call_emergence_react`）** 的三轮调用，在 `l2_engine.py:766-835`：

```
Round 1: LLM + 探索工具 → check_inbox（查看收件箱）
Round 2: LLM + 探索工具 → check_inbox（基于工具结果继续探索）  
Round 3: LLM 无工具 → 输出内心独白文本
```

是一个**小 ReAct 循环**（最多 2 轮工具调用），让 agent 在写内心独白前先探索：查收件箱、展开 DAG、搜索记忆等。三轮结束后，从文本中分离 `---DOUBT---` 块得到自我不确定感。

而日志里没显示的**意图决策**（`_call_intent_react`）是另一个独立 LLM 调用，走的是 `l2_agent.react_nodb(silent=True)`，LLM 日志被静默了。

所以每次 L2 加柴实际上是 **4 次 LLM 调用**：1 次意图 + 3 次涌现。
