Rich Markdown 渲染和 logger 输出**同时竞争终端**。Rich 用 `Console.print()` 写 stdout，logger 写 stderr，两个独立缓冲。当 LLM 生成的主动消息内容较长（"直接输出你想对对方说的话..." 这段本身就是 LLM 生成的），Rich 分段渲染时 logger 的输出插入进来，造成错行。
