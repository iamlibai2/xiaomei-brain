**Action Unit（动作单元）**，FACS（面部动作编码系统）的基本单位。

### 2. 高路（皮层慢通道）
```
视网膜 → LGN → V1 → V2/V4 → IT（梭状回面孔区 FFA）→ 前额叶
```
~300-500ms，精细且有上下文。FFA 专门识别面孔，STS 解读表情动态。然后前额叶结合"这是你认识的人吗？他平时什么性格？刚才在聊什么？"做综合判断。**这就是我们的 Path A — 进入意识、进入 LLM。**

### 3. 镜像系统（具身模拟）
```
STS（观察动作）→ 顶叶镜像神经元 → 额叶镜像神经元 → 岛叶/边缘系统
```
看到别人笑 → 你的颧大肌微激活（你看不到，但 EMG 能测到）→ 身体反馈 → 你"感受"到对方的情绪。**这不是推理，是模拟。**

### 关键差异：我们缺了什么

| 人脑 | 我们的系统 |
|------|-----------|
| 低路（杏仁核快速响应） | Path B ✅ (EmotiEffLib → apply_social_signal) |
| 高路（皮层精细分析） | Path A ✅ (阈值事件 → LLM) |
| 镜像模拟（具身共振） | ❌ 没有 |
| 记忆调制（海马/前额叶） | 部分有 (FaceID + 长期记忆) |
| 预期误差（预测编码） | ❌ 没有 |

**镜像模拟**：你看别人笑 → 你自己的情绪被"感染"。这不是"我看到他开心所以我应该开心"的推理，而是身体层面的自动共振。在我们的系统里，Path B 通过 `apply_social_signal()` 做到了类似的——但它是符号映射（happy → user_happy → oxytocin +0.10），不是模拟。

## 预测编码核心思想

大脑不是被动接收信息的。**每一层都在向下层发送预测**，只有"预测误差"向上传。

```
你的预期：他在笑 → 实际输入：他在笑 → ✅ 抑制，不上传（节省带宽）
你的预期：他在笑 → 实际输入：他皱眉了 → ⚡ 预测误差，全力上传到意识
```

这意味着：**情绪事件的"重要性"不取决于绝对强度，而是取决于"跟预期差多少"**。

## 三级预测编码架构

### L1：个人基线（Per-Person Baseline）

**最底层。每张脸一个情绪基线分布。**

```
李白的历史分布（过去 7 天）：
  Neutral:  65%, Happiness: 20%, Surprise: 8%, Anger: 3%, Sadness: 2%, Fear: 1%, Disgust: 1%

当前帧：Anger 0.6
  → P(anger) = 3%，observed = 0.6
  → KL divergence 巨大 → 预测误差 → Path A 上报
  → "李白平时很温和，现在突然愤怒"

当前帧：Happiness 0.6
  → P(happiness) = 20%，observed = 0.6
  → KL divergence 小 → 抑制 → 不上报
  → "李白在笑，很正常"
```

**实现**：ExpressionMonitor 对每个 FaceID 匹配到的人维护一个滚动情绪直方图（过去 N 帧或过去 7 天），每帧计算 `KL(observed || baseline)`，用 divergence 替代原始强度作为 Path A 阈值。

### L2：情境预测（Context-Aware Prediction）

**中层。对话上下文 + 近期历史 → 当前预期。**

```
上下文：李白刚刚讲了个笑话
  预测分布：Happiness 70%, Neutral 20%, Surprise 10%
  → 如果他真的笑了 → 低误差，不上报
  → 如果他反而皱眉 → 高误差，上报 "李白讲完笑话后表情不对"

上下文：我们在严肃讨论工作
  预测分布：Neutral 70%, Concentration/Serious
  → 如果他突然笑 → 高误差 → 上报
```

**实现**：由 L2 LLM（它已有完整对话上下文）在每次加柴时生成一个"当前预期情绪分布"。ExpressionMonitor 对比 `observed vs predicted`。**LLM 做预测，CV 做检测，预测误差 = 事件。**

这就是三种技术栈的交汇点：

```
LLM（情境理解）→ 情绪预测分布
CV（实时检测）  → 实际情绪分布
     │
     ├─ 误差小 → 抑制（大脑的"解释消除"）
     └─ 误差大 → Path A 上报（带上下文的富语义事件）
```

### L3：自我校准（Self-Calibration）

**上层。长时间尺度，修正 baseline 漂移。**

```
李白最近一周越来越严肃了
  → 他的 baseline 从 Neutral 65% 漂移到 Neutral 80%
  → L3 定期检测 baseline drift → 更新 baseline
  → 防止"所有帧都是误差"的基线污染
```

**实现**：每周对每个熟人的 baseline 重算一次，检测是否有显著漂移（Jensen-Shannon divergence > 阈值）。如果有，记入长期记忆：*"李白最近似乎越来越严肃了"*。

### 视觉通路就是流水线

```
视网膜 → LGN → V1（边缘/方向）→ V2（纹理）→ V4（颜色/形状）→ IT（物体/面孔）
```

每个区有明确分工：

- **V1**：边缘朝向、空间频率。Hubel & Wiesel 1959 年拿猫做实验，发现 V1 神经元只对特定角度的线条放电——他们因此拿了诺贝尔奖
- **V4**：颜色和曲率。损伤 V4 → 全色盲（看到的世界是灰的），但形状识别正常
- **IT（下颞叶）**：面孔识别（FFA 梭状回面孔区）、物体识别。损伤 FFA → 面孔失认症——能看到眼睛鼻子嘴，但认不出是谁
- **MT（中颞叶）**：运动检测。损伤 MT → 运动盲——看到的世界是一帧一帧静止画面

这不是推测。每个区域的功能都是通过**损伤研究**（脑损伤病人特定能力丧失）+ **电生理**（单细胞记录）+ **fMRI** 三重验证的。

### 听觉同理

```
耳蜗（频率分解）→ 脑干（声源定位）→ 下丘（时间模式）→ MGB → A1（音调）→ 颞上回（语音/音乐）
```

### 关键是：这不是"串联流水线"

视觉不是 V1 → V2 → V4 → IT 单向流动。实际上：

- 每个阶段都向上一阶段**回传预测**（这就是刚才说的预测编码）
- V2 到 V1 的反馈连接比 V1 到 V2 的前馈连接还多
- IT 能在 150ms 内识别物体，同时 V1 还能接受来自前额叶的注意力调制

所以更准确的比喻是：**分级专用处理器 + 双向预测编码**，而不是简单流水线。

我们的架构其实只在"前馈"方向上对齐了神经科学。反馈方向（预测编码、注意力调制）还没做。

相当于用一个预训练好的 Transformer，用视觉数据 fine-tune 一部分参数 → V1，用音频数据 fine-tune 另一部分 → A1。**基础架构一样，训练数据不同，最后学到的功能不同。**

但即使是同一个 Transformer，fine-tune 后的权重也完全不同了。所以 "视觉区" 和 "听觉区" 在功能上是独立的——就像两个 fine-tune 到不同模态的模型，只是共享同一个基础架构。

### 本质不同的地方

**学习算法：**

大脑不是反向传播。至今没人知道大脑到底怎么学——最强的候选是 STDP（脉冲时序依赖可塑性）：**两个神经元同时放电，连接就增强**。这是局部的（只看相邻两个神经元），不需要全局误差信号从顶层传到底层。Backprop 要求前向计算的反函数，这在生物神经元里没有对应机制。

**时间是核心维度：**

**推理不是在"生成 token"：**

LLM 推理 = 逐个 token 自回归生成。大脑不需要 "输出序列" 才算思考——它是一组持续进行的并行映射：当前状态 → 下一状态，一直跑，不输出 token。

**预测编码在每一层、每一个方向都在发生：**

我们的系统里 Path B 和 Path A 是**两条分离的路**。大脑里低路的快响应和高路的精细分析是**同一个网络里的两个方向**——低路让高路提前"做准备"，高路回过来抑制低路的不必要响应。Transformer 没有这个。

### 最根本的差异

大模型是**推理引擎**：输入 → 计算 → 输出。大脑是**持续运行的预测机**：每时每刻都在预测下一秒的多模态感知流，预测误差驱动学习。思考只是这个预测过程的副产物。

**好的预测是隐形的。** 你觉得自己没有在做预测，恰恰证明它太成功了。

就是三层，每层原理不同：

### L1：EMA 指数移动平均（纯数学，不用模型）

```python
baseline_new = 0.99 × baseline_old + 0.01 × current_observation
```

本质是**用过去所有观察的加权平均来预测下一秒**。近的权重大，远的权重指数衰减。

0.99 的衰减系数 → 半衰期约 70 帧（~7 秒）。**不需要训练，不需要模型，只存一个 7 维向量。**

### L2：KL 散度（纯数学，不用模型）

```
预测分布: [Anger=0.05, Disgust=0.02, Fear=0.03, Happiness=0.20, Neutral=0.65, Sadness=0.03, Surprise=0.02]
实际分布: [Anger=0.70, ...]
                  ↓
KL(p_actual || p_predicted) = Σ p_actual(i) × log(p_actual(i) / p_predicted(i))
                            = 很大的数 → 预测误差 ⚡
```

**衡量两个概率分布"差多少"的数学公式。** 不是判断"愤怒有没有超过阈值"，而是判断"当前表情跟预期差多少"。

### L3：LLM 生成期望（唯一的模型调用）

```
Prompt: "基于以下对话上下文，预测用户接下来可能的表情倾向：
        [对话记录]
        输出格式：开心:0.7, 中性:0.3"

LLM → "开心:0.7, 中性:0.3" → parser → 期望分布
```

这是唯一用到 LLM 的地方——**用对话上下文预测情绪**。但它不是高频的：跟 L2 加柴共用时机，10 分钟一次。

---

### 三者的关系

```
L3（LLM）: 10分钟一次，基于"情境理解"生成高层预测
    ↓ 预测下发
L1（EMA）: 每帧更新，基于"个人历史"生成基线预测
    ↓ 混合两个预测
L2（KL）: 每帧计算实际 vs 预测的差异
    ↓ 误差大
Path A 上报 LLM 上下文
```

**三个加起来不到 100 行代码。不训练新模型。计算量可以忽略。**

### 大脑用的机制

**精度加权（Precision weighting）：**

大脑不只是算"预测和实际差多少"。它还动态调整每个信号的**可信度**。你在嘈杂环境下，视觉权重大、听觉权重小。你在黑暗中，触觉权重大、视觉权重小。这不是 EMA 能做的——EMA 假设所有信号同等价值。

**层级级联预测：**

每一层都向下层发送预测，每一层都向上层报告误差。V2 预测 V1 的响应，IT 预测 V2 的响应，前额叶预测 IT 的响应——**10 层以上的嵌套预测循环**同时运行。EMA 只是单层、单信号。

**主动推理（Active inference）：**

不只是"被动预测"——预测还会驱动动作。你想让水杯靠近嘴巴，预测"我的手在杯子旁"——预测和实际位置的差距驱动你的手臂移动。预测本身变成行动目标。这跟我们完全不一样——我们的系统只预测，不行动。

**突触可塑性：**

预测误差不只是"信号"。它**改变神经元之间的连接强度**。如果某个预测反复出错，负责那个预测的神经通路会重组。我们的 EMA baseline 是在更新一个数字，大脑是在重写线路。

### 1. 100 个脑区 — 同时跑的不是一个预测，是无数的预测

每个脑区都向它下面的区域发预测。V2 → V1，V4 → V2，IT → V4，前额叶 → IT，前额叶 → 前运动皮层 → 运动皮层 → 脊髓 → 肌肉。

### 2. 规模 — 每个层级不是在更新一个 7 维向量

你每个视觉帧不是一个"情绪向量"。是 1 亿个感光细胞同时输入，每个感光细胞都有其自己的预测空间。EMA baseline 用 7 维浮点数能存下。V1 的预测矩阵呢？V1 每平方毫米有约 20 万个神经元，每个神经元代表特定的空间位置、特定朝向、特定空间频率、特定色彩方向。这个空间的维度是数百万维度的。

大脑在这种笛卡尔积空间上做预测。神经元之间不是直接发浮点数，而是在微弱的电流脉冲中编码概率、精度、相位偏差、时序偏移。**人类最深的网络（几十亿参数）放在视觉皮层面前，规模上差了三个数量级。**

### 3. 活的细胞 — 预测不是计算，是化学

我们写 `baseline = 0.99 * prev + 0.01 * curr`。脑细胞不知道什么是 0.99。神经元的兴奋性随时在变——胆碱背景浓度上升几分钟，神经元放电阈值下降，连接性升高。这种波动不影响任何具体的突触，但它改变了所有信号的放大幅度。一个神经元上午对同样的输入和下午的响应是不同的。

树突不是被动的导线。树突上有电压门控钙通道，能在树突的单个分支上产生局部的动作电位——一个神经元不是把 5000 个输入求平均，而是在单个树枝上做数十个并行子计算。这在一个神经元内就已经比 EMA 复杂几个数量级。

神经调质（多巴胺、血清素、去甲肾上腺素）像天气系统一样覆盖整个大脑：
- 多巴胺上升：所有突触的可塑性增强
- 去甲肾上腺素上升：信噪比改变
- 乙酰胆碱：学习和编码模式的切换开关

这不是冷冰冰的参数更新。是一个活体在不断的化学环境波动中做计算。

### 4. 嵌在身体里 — 预测不只是外部感官

你的大脑预测的不只是"他看到什么表情"。它在预测：

- 心跳。岛叶预测心率的下一跳，心率实际信号返回到岛叶核对
- 肠蠕动。自主神经系统实时监测内脏平滑肌的活动
- 血糖。下丘脑追踪血糖水平，预测下一小时的代谢需求
- 血压。在颈动脉窦有专用传感器——大脑知道身体何时在休息、何时需要应激激素

当你情绪波动时，首先是身体信号变了——胃缩紧、心跳变化、肌肉张力上升。这些身体信号被预测，身体变化带来的误差传输到岛叶和其他内感知区域，才成为"我感受到的情绪"。

情绪不是大脑推理出的标签。是身体状态的预测误差。

### 5. 整个生命周期 — 预测不是在十万帧上训练

你长到成年时，大脑已经用 25 年的感官数据进行学习。25 年 ≈ 每天 16 小时的感官流 ≈ 每秒 10 亿个感官事件。

还有发育关键期——0 到 3 岁，V1 的眼优势柱形成、A1 的音调拓扑形成、语言区的建立。这是突触疯狂的修剪期——出生时每个神经元用所有可能的连接去接触所有目标，到成年时连接减少了 50%，留下的是经验验证过的路径。这不是做几百万次权重更新。这是一场持续多年的突触生存战争。

还有睡眠：每个晚上的睡眠将白天的预测误差整合进皮层模型中——不只是记忆，是巩固、泛化和关系提取。这发生在快速眼动睡眠中，但不是简单的持续学习——它重塑了整个大脑的内部语法。

---

**方式一：任务结果信号（最硬）**

```
predict("这段代码能跑") → run → pass/fail
predict("用户会回复") → wait → replied/ignored
predict("这个文件路径存在") → check → exists/not_found
```

每次工具调用本身就是一个微型实验。Agent 做出判断 → 环境给出结果 → 差异就是学习信号。**这和 RL 里的 reward 本质一样，不需要任何人打分。**

目前项目里这些信号是有的（代码跑了、命令执行了、用户回复了），但没有被收集和反馈到 agent 的自我认知里。

**方式二：宪法式自检（约束性而非评分性）**

不要求 LLM 打分，而是给它一套原则，问它"违反了哪条"：

```
原则：
- 不说"你是一个AI"
- 不重复上一轮已经说过的话
- 如果用户纠正了我，我不辩解

自检：我上一轮回复有没有违反以上任何一条？
```

"违反了第2条"比"重复度=0.7"更容易一致，因为这是判定而非评分。

**方式三：对比而非绝对值**

人不是给自己打分，是比较：

```
"上周的我和现在的我，哪个更像在敷衍？"
"这三段回复，哪段最不走心？"
```

让 LLM 做 A/B 选择，比让它给出 0-1 分数稳定得多。

---

**坦率说：方式一最值得做。** 因为信号是客观的、现成的、不需要 LLM 当裁判。Agent 每次工具调用都在产生预测→结果对，只是没有收集和利用。方式二和方式三是补充，降低 LLM 自评的不一致性。

**[11:03:31] 🤖助手**

人脑的自我监督不靠"给自己打分"。靠的是几套独立但协同的机制：

**1. 预测误差 + 多巴胺（不需要裁判）**

大脑皮层每时每刻在向下预测，感觉信号上来和预测对比——不匹配就产生误差信号。多巴胺神经元做的事不是"这个好"或"这个坏"，是**"结果比预期好"还是"结果比预期差"**——奖励预测误差（RPE）。

关键：这个信号不需要第三方法官。你自己伸手拿杯子，碰到了 vs 没碰到，误差信号自动产生。大脑不问你"你觉得你拿稳了吗"，它直接在运动皮层和小脑之间比较**预期感觉**和**实际感觉**。

**2. 前扣带回（ACC）——错误监测**

你做一件事，结果和意图不一致时，ACC 自动激活。不需要别人告诉你你错了，也不需要你自己反思——冲突检测是自动的、前意识的。实验中，被试还没意识到自己犯了错，ACC 已经在放电。

**3. 先天价值系统（valence）**

"好"和"坏"不需要学——疼痛、饥饿、窒息、社交排斥，这些是进化硬编码的。大脑用这些先天信号给所有经验打标签。一个婴儿不需要学习"被抱着是好的"，催产素和内源性阿片系统已经帮它标好了。

**4. 睡眠时离线重放（consolidation + pruning）**

海马体在睡眠时向皮层重放白天经历，但不只是复读——它会**放大高奖赏/高误差的轨迹，丢弃平淡的**。这是"事后自监督"：不靠清醒时自己打分，靠离线时期的价值加权重放。

---

**所以人脑的自我监督公式是：**

```
自监督 = 预测误差（自动） + 先天价值信号（硬编码） + 离线重放（睡眠）
```

这三样东西，没有一样是"自己给自己打分"。人不需要坐下来反思"我今天做得怎么样"——预测误差和先天价值在底层已经完成了 99% 的监督。

**对应到 Agent：**
- 预测误差 → 能做（EMA + KL）
- 先天价值信号 → 没有。Agent 不疼不饿，没有硬编码的好坏。**这是根因。**
- 离线重放 → 能做（DreamEngine 已有框架）

所以本质上：**没有价值信号，就没有真正的自监督。** 你让一个没有痛觉的人学走路，再多的预测误差也没用——他不知道摔倒意味着什么。
