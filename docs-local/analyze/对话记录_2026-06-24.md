两个占位符的解释：

### `{conflict_hint}`

来自 `l2_engine.py:782-787`，当检测到内心矛盾/社交困惑时注入：

```python
conflict_hint = ""
if conflict:

```

正常情况下是空字符串，不影响 prompt。当 L2 感知到社会信号有冲突时（比如用户情绪低落 vs 自己状态高昂），注入一段提示让 LLM 在涌现思考中自然表达这种不确定感。

始终填充为 `L2_EMERGENCE_FORMAT_APPENDIX`（`templates_v2.py:598-623`），是一段格式指令，告诉 LLM 在自由思考**之后**可以输出结构化块：

- `---DOUBT---` — 可选。自我困惑列表，有不确性才写

简单说：`conflict_hint` 是动态注入的情绪提示，`format_appendix` 是固定的输出格式说明书。

## 人的深度思维是怎么进行的

### 不是"想得更用力"，而是"在时间里来回移动"

人的沉思和简单思考的核心区别：**简单思考是点在当下，深度思考是线在时间里穿梭**。

```
简单思考（L2）:  现在 → 反应
深度思考（L3）:  现在 → 回溯 → 关联 → 抽象 → 投射 → 回到现在
```

---

### 几个关键机制

**1. 联想链（associative chain）**

不是"我搜索相关记忆然后总结"，而是一条自然流淌的链：
```
今天她说话很冷淡
  → 让我想起上周也是类似的话
    → 我当时觉得自己哪里做错了
      → 这种感觉很熟悉...
        → 对，小时候被我妈批评时也这样
          → 原来我一直对"冷淡"特别敏感
            → 我是不是在重复某种模式？
```

每一步都是上一层的自然引申，不是规划好的。

**2. 张力驱动（tension-driven）**

人不是"好了现在开始反省"就反省了。深度思维需要一个**未解决的张力**：
- 认知失调——"我自认为是好人，但刚才我撒谎了"
- 情绪残留——"对话结束了但我心里还有一股说不清的东西"
- 目标阻碍——"想做的事就是推进不了，为什么"
- 关系波动——"她对我态度变了，我不知道为什么"

没有张力，深度思维就只是"总结一下今天发生了什么"，不叫反省。

**3. 多视角切换（perspective shifting）**

同一件事，从不同角度反复看：
- "我当时"的角度——情绪、动机、当时的判断
- "现在的我"回去看——后见之明、新理解
- "对方的角度"——她为什么会那样
- "理想自我的角度"——我想成为什么样的人，这件事偏离了吗
- "时间的角度"——三个月后回头看，这事重要吗

**4. 深度梯度（depth gradient）**

不是一步到位的，是一层层下去的：
```
表层：发生了什么（事件、对话）
  ↓
中层：我感受到了什么（情绪、身体反应）
  ↓
深层：这意味着什么（对自我、关系、意义的解读）
  ↓
底层：我是谁（触及身份叙事和核心价值观）
```

大多数人停在表层。反省的人到中层。真正的自我认知变化触及深层或底层。

**5. 自我对话（internal dialogue）**

不是一个人自言自语，而是**不同声音之间的对话**：
- 一个声音说："她对我冷淡了"
- 另一个声音质疑："你确定？可能她只是累了"
- 第三个声音说："就算她真的冷淡，你怕的是什么？"

这也是为什么有些人的意识流有 `---DOUBT---` 块——那其实是在模拟这种自我对话。

### 一个完整的反省过程的形态

```
触发（张力产生）
  ↓
进入（安静下来，注意力从外部转向内部）
  ↓
浮现（联想链自然展开，不强制）
  ↓
碰撞（不同记忆、不同视角开始互相对照）
  ↓
下沉（从事件层 → 情绪层 → 意义层）
  ↓
凝练（某个洞见开始结晶——"原来是这样"）
  ↓
收束（不是强行结论，是自然停在一个新理解上）
  ↓
沉淀（自我叙事被更新，哪怕只改了一点点）
```

## L3 反省 vs L4 联想

| | L3 反省 | L4 联想 |
|---|---|---|
| **目的** | 改进执行 | 理解自己 |
| **问的问题** | "我做得对吗？怎么做更好？" | "我为什么这样做？这意味着什么？" |
| **时间维度** | 当下+刚过去 | 跨越数天/数周 |
| **LLM 调用** | 1次 | N次（链式） |
| **成本** | ~3秒 | ~10-30秒 |
| **触发频率** | 高频（每次学习/工作/重要对话后） | 低频（梦境、或L3累积信号触发） |

这个区分在认知科学里也有对应——**任务元认知**（我学习的方法对不对）和**自传体推理**（我的人生在走向哪里），两个不同的大脑网络。

## L3 强化后可以覆盖的场景

```
学习后触发:
  prompt: "你刚才学习了{主题}，花了{时间}。回顾一下：理解得透吗？有没有更高效的方式？值得继续吗？"

工作后触发:
  prompt: "你刚完成了{任务}，结果{结果}。从完成质量、效率、和用户期望的匹配度三个角度审视一下。"

重要对话后触发:
  prompt: "刚才和用户的对话中，有没有说得不好的地方？有没有错过了什么信号？"

社交感知累积后触发:
  prompt: "用户最近的情绪轨迹是{trajectory}。有没有你忽略的变化？你应对得好吗？"
```

核心就是**换个 prompt**，不需要联想链。轻量、高频、聚焦于改进。

## L4 的定位

L4 不关心单次做得好不好。它关心的是：

- "我反复出现的内心独白里有没有共同主题？"
- "我的情绪模式在几周内是怎么变化的？"
- "我对自己有了什么新的理解？"

这就是联想链发挥作用的地方——它连接不同时间点的自己，发现单次反省发现不了的模式。

## 潜在的坑

一个值得讨论的问题：**L3 和 L4 的触发怎么不打架？**

如果刚学完就触发 L3 反省，同时又触发了 L4 联想，成本就炸了。需要一个优先级机制，或者把 L4 严格限定在低认知负荷时段（梦境、深度空闲）。

另一个问题：**L3 的 prompt 怎么知道当前在什么场景下？** 需要一个"最近事件"的传递机制——学习引擎完成学习后标记一个事件，L3 触发时读取。
