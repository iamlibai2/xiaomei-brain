```python
def _should_social_cognition(self, agent_state):
    # 1. SLEEPING/DREAMING 中不触发
    if agent_state in ("sleeping", "dreaming"):
        return False
    # 2. 能量 < 0.25 不触发
    if si.body.energy < 0.25:
        return False
    # 3. 冷却中不触发（sc_cooldown=900s）
    if elapsed < 900:
        return False
    # 4. 对话驱动：最近对话 >= 200 字才触发
    recent = self._get_recent_conversation()
    if recent and len(recent) >= 200:
        return True
    # 5. 定期兜底：即使没对话，3600s 后也触发一次
    if elapsed >= 3600:
        return True
    return False
```

核心逻辑：**对话驱动**为主——没对话或对话太短（< 200 字）就不触发，避免 LLM 对着不存在的社交内容空评论。3600 秒定期兜底保证不会永远不触发。
