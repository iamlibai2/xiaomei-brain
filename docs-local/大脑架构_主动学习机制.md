# 大脑架构设计：主动学习机制

> 创建时间：2026-04-17
> 状态：讨论完成，待实现

---

## 概述

主动学习是跨多个层的协作机制，不属于单一层。

```
主动学习涉及：
- Metacognition（识别知识缺口）
- Purpose（创建学习目标）
- Basal Ganglia（选择学习行动）
- Agent（执行：搜索、阅读）
- Drive（好奇心驱动）
- Memory（存储知识）
```

---

## 运行流程

### 触发：怎么知道要学习？

```
情况 1：用户直接要求
    用户："帮我查一下量子计算"
        ↓
    → 直接触发学习

情况 2：Metacognition 发现知识缺口
    Agent 在执行任务时
        ↓
    Metacognition 发现："这个问题涉及量子计算，但我不懂"
        ↓
    → 标记知识缺口
        ↓
    Drive 认知欲被激活
        ↓
    → 触发学习

情况 3：Drive 好奇心驱动
    认知欲持续累积，达到阈值
        ↓
    "最近总是遇到 AI 相关的问题，我好奇 AI 是怎么实现的"
        ↓
    → 主动触发学习（不需要外部要求）
```

---

## 学习过程

```
触发学习
    ↓
创建学习目标：比如"了解量子计算"
    ↓
分解目标：
    1. 搜索量子计算基础概念
    2. 阅读搜索结果
    3. 提取核心知识
    4. 存储到记忆中
    ↓
Agent 执行：
    工具：百度搜索
        ↓
    获取：搜索结果列表
        ↓
    判断：是否需要深入阅读某个页面
        ↓
    如需深入：
        工具：阅读网页
            ↓
        获取：网页内容
            ↓
        提取：关键信息
            ↓
    整理：形成简短的知识点总结
        ↓
存储到 Memory
    ↓
更新 Metacognition 的知识状态：
    "现在我知道量子计算是什么了"
```

### 第一步：Metacognition 发现知识缺口

```python
# 现有 check() 可以扩展
def check(self, goal: Goal, context: Context) -> Optional[Adjustment]:
    # 现有逻辑：检查能否完成目标
    can_achieve, confidence = self._evaluate_ability(goal, context)

    # 新增：检查是否涉及未知领域
    unknown_topics = self._detect_unknown_topics(goal, context)
    # 调用 LLM：问"这个目标涉及哪些你不懂的概念"
    # 返回：["量子计算", "纠缠态"]

    if unknown_topics:
        # 标记知识缺口
        self.status.uncertainty_topics = unknown_topics
        return Adjustment(
            type=ADJUSTMENT_TYPE.REQUEST_LEARNING,
            reason=f"涉及未知领域：{unknown_topics}",
            suggested_action="我需要先了解一下这些概念"
        )
```

## 关键问题

### 1. 学习什么时候发生？

```
选项 A：立即
        → 发现知识缺口 → 马上学习 → 再继续原任务
        → 缺点：打断当前任务

选项 B：后台学习
        → 先继续原任务
        → 后台启动学习
        → 学习结果存储，但不立即影响当前任务
        → 优点：不打断

选项 C：批量学习
        → 先记录所有知识缺口
        → 当前任务完成后
        → 批量学习所有缺口
        → 优点：效率高
```

### 2. 学习深度如何确定？

```
选项 A：浅度学习
        → 搜索结果摘要
        → 知道"量子计算是一种..."

选项 B：深度学习
        → 阅读多篇文章
        → 理解原理

选项 C：按需学习
        → 用户问浅问题 → 浅学
        → 用户问深问题 → 深学
```

### 3. 学到的知识存在哪？

```
长期记忆（Memory）
    ↓
命名空间：knowledge/physics/quantum_computing
    ↓
存储内容：
    - 量子计算是什么
    - 基本原理（简化版）
    - 应用场景
    - 相关人物/项目
    - 来源链接（可追溯）
```
