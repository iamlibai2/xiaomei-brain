● xiaomei-brain 项目结构

  xiaomei-brain/
  │
  ├── src/xiaomei_brain/              # 主源代码
  │   │
  │   ├── 核心 (Core)
  │   │   ├── agent.py               # Agent 核心（LLM + tools 执行引擎）
  │   │   ├── agent_manager.py       # 多 Agent 管理（xiaomei/xiaoming/default）
  │   │   ├── config.py              # 配置解析（OpenClaw JSON 格式）
  │   │   ├── config_provider.py     # 配置 Provider（get/patch/apply/hash）
  │   │   ├── llm.py                 # LLM 客户端（支持多 Provider）
  │   │   └── context.py             # 上下文管理
  │   │
  │   ├── 记忆 (Memory)
  │   │   ├── memory/store.py        # 长期记忆存储
  │   │   ├── memory/episodic.py     #情节记忆
  │   │   ├── memory/conversation.py # 对话记录
  │   │   ├── memory/dream.py        # 梦境处理
  │   │   ├── memory/search.py       # 向量搜索
  │   │   ├── memory/layers.py       # 工作记忆层
  │   │   └── memory/scheduler.py    # 记忆调度
  │   │
  │   ├── 工具 (Tools)
  │   │   ├── tools/registry.py      # 工具注册表
  │   │   └── tools/builtin/         # 内置工具
  │   │       ├── shell.py          # Shell 命令
  │   │       ├── file_ops.py       # 文件操作
  │   │       ├── memory.py         # 记忆工具
  │   │       ├── tts.py            # TTS 工具
  │   │       ├── music.py          # 音乐工具
  │   │       ├── image.py         # 图像工具
  │   │       ├── websearch.py     # 搜索工具
  │   │       └── webget.py        # 网页获取
  │   │
  │   ├── 对话与主动 (Chat & Proactive)
  │   │   ├── session.py            # 会话管理
  │   │   ├── proactive.py          # 主动触发引擎
  │   │   └── reminder.py          # 提醒管理
  │   │
  │   ├── 语音 (Speech)
  │   │   ├── speech/tts.py        # TTS 文字转语音
  │   │   └── speech/music.py      # 音乐生成
  │   │
  │   ├── 渠道 (Channels)
  │   │   ├── channels/gateway.py  # 消息网关
  │   │   ├── channels/feishu.py    # 飞书频道
  │   │   ├── channels/dingtalk.py # 钉钉频道
  │   │   └── channels/wechat.py   # 微信频道
  │   │
  │   └── WebSocket 服务器
  │       ├── ws/server.py         # FastAPI + WebSocket 入口
  │       ├── ws/protocol.py       # 消息协议定义
  │       ├── ws/session.py        # 客户端会话
  │       └── ws/connection.py     # 连接管理
  │
  ├── examples/
  │   ├── basic_agent.py          # 命令行 AI 对话（单用户本地测试）
  │   └── ws_server.py            # WebSocket 服务器（网络 API）
  │
  ├── bak/                        # 废弃文件备份
  │
  └── config.json                 # 主配置文件

  运行模式

  ┌─────────────────────────────────────────────────────────────┐
  │  模式 1: basic_agent.py (命令行对话)                         │
  │  ┌──────────────┐                                           │
  │  │ 终端输入/输出 │ → Agent → LLM                            │
  │  └──────────────┘                                           │
  └─────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  模式 2: ws_server.py (WebSocket 服务器)                      │
  │  ┌──────────────┐                                           │
  │  │ 0.0.0.0:8765 │ → /ws → ws_endpoint → Agent → LLM         │
  │  └──────────────┘                                           │
  └─────────────────────────────────────────────────────────────┘

  多 Agent 架构

  AgentManager (agent_manager.py)
  ├── agents/
  │   ├── default/    # 默认助手
  │   │   ├── talent.md
  │   │   ├── memory/
  │   │   └── sessions/
  │   ├── xiaomei/   # 小美
  │   │   ├── talent.md
  │   │   ├── memory/
  │   │   └── sessions/
  │   └── xiaoming/  # 小明
  │       ├── talent.md
  │       ├── memory/
  │       └── sessions/

  WebSocket 消息流

  客户端 ──WebSocket──→ ws_endpoint (server.py)
                           │
                           ├── session_start ──→ 创建会话
                           ├── chat ──────────→ Agent.stream()
                           │                      │
                           │                      ├── tools.execute()
                           │                      └── text_chunk (流式返回)
                           ├── ping ───────────→ pong
                           └── JSON-RPC ───────→ config.get/patch/apply

  配置系统

  config.json (OpenClaw 格式)
  ├── meta
  ├── models (LLM Provider 配置)
  ├── agents.list (多 Agent 配置)
  ├── channels (飞书/钉钉等)
  ├── bindings (路由规则)
  └── xiaomei_brain (xiaomei-brain 特定配置)
