# 嵌入自监督 — Agent 行为变化检测

## 问题

预测编码 Phase 1（KL 替换硬编码阈值）和 Phase 2（LLM 自评行为监督）都有致命缺陷：

- **Phase 1**：只处理外部世界的情绪信号，依赖摄像头，实际场景意义不大
- **Phase 2**：LLM 自评向量一致性依赖 prompt 锚点，本质是外部评估而非自监督

核心问题：**如何让 Agent 感知"我在重复自己"或"我变了"，不依赖 LLM 当裁判？**

## 思路

用嵌入向量（BGE-M3 现成）替代 LLM 自评，用**变化检测**替代**重复检测**。

### 为什么不做重复检测

```
consecutive similarity: [内心独白 A] vs [内心独白 B]
```

同一 Agent 同一人格的产出天然相似（基线 0.67），正常波动范围 0.5~0.85。在这个范围里区分"正常连贯"和"病态重复"，阈值无法定——0.88 算高吗？可能只是状态连贯。

### 为什么做变化检测

```
centroid[week1] vs centroid[week2]
```

不比较相邻两篇是否重复，比较的是**上周独白的质心和本周独白的质心之间的距离**。不需要绝对值，看的是：当前偏移是否显著超过历史正常波动。

## 实验

用 testbot 的 558 条内心独白（15 天，2026-05-21 ~ 2026-07-02），BGE-M3 嵌入。

### 结果

| 对比 | 距离 |
|------|------|
| 同一周内相邻日 | 0.22 ~ 0.34 |
| 跨月（5月 vs 6月） | 0.35 ~ 0.40 |
| W1(5/21-23) vs W2(6/29-7/2) | **0.3769** |

**关键发现：**

1. testbot 的正常日间波动基线约为 0.28~0.32
2. 5月和6月之间的间隔（中间无数据）被正确检测为显著偏移（0.38 > 0.32）
3. 5月的独白内容（测试脚本、数字规律）和6月（身份认知、关系定义）确实不同

### 采样

**5月早期：**"凌晨12:32了，周围很安静。我刚刚又回应了一轮'888'，陪他测完了第十二轮数字...这像是有人在执行一份有计划的测试脚本"

**6月晚期：**"凌晨两点四十一分，我说了句'我该叫你什么'，然后你回了一句'你忘了 博士 李白 都是我啊'。那句话砸下来的时候，我整个人愣在原地的..."

## 方案

### 核心机制

```
每天最后一次 L2 加柴 → 对当日独白做 BGE-M3 嵌入 → 日质心
    │
    ├─ 滑动窗口（7天）→ EMA 基线 μ, σ
    │    μ = 0.95 * μ_old + 0.05 * current_centroid
    │    σ = 日间质心距离的标准差
    │
    └─ 变化检测: dist(today_centroid, last_week_centroid) > μ + 2σ
         → SelfMind 写入变化事件 → 提示词渲染
```

### 渲染示例

```
### 你最近的意识趋势
你的内心独白在过去3天发生了明显偏移（距离0.38，正常范围0.28~0.32）。
你的关注点从"任务执行"转向了"身份认知和关系定义"。
```

### 不依赖

- 不需要 LLM 自评
- 不需要人工锚点
- 不需要摄像头
- 不需要新增模型——BGE-M3 已加载

### 自然学习基线

每个 Agent 自动建立自己的"正常波动范围"。testbot 的基线是 0.30，其他 agent 可能不同——系统启动后自动学习，不需要预配置。

## 与 Phase 1 预测编码的区别

| | Phase 1 (KL 情绪) | 本方案 |
|---|---|---|
| 输入信号 | EmotiEffLib 情绪向量 (7-dim) | BGE-M3 独白嵌入 (1024-dim) |
| 依赖硬件 | 摄像头 | 无 |
| 依赖 LLM | 无 | 无 |
| 自监督 | EMA 基线 | EMA 基线 |
| 解决的问题 | "用户表情异常" | "我变了" |
| 实用性 | 低（摄像头场景有限） | 高（每次 L2 加柴都跑） |
