> 创建时间：2026-04-23

## 1. 初始化流程

```
ConsciousLiving.__init__()
    │
    ├── DriveEngine 初始化
    │   ├── 加载 drive_config.yaml（欲望阈值）
    │   ├── 加载 drive_state.json（恢复状态）
    │   └── 初始化 Emotion/Hormone/Motivation/Desire
    │
    ├── PurposeEngine 初始化
    │   ├── 加载 identity.md（存在意义）
    │   ├── 加载 goals.json（恢复目标树）
    │   ├── 创建战略目标（meaning-root）
    │   └── 传入 Drive 引用（完成目标时通知）
    │
    ├── Consciousness 初始化
    │   ├── 接收 Drive 引用
    │   ├── 接收 Purpose 引用
    │   ├── 初始化 SelfImage（6模块）
    │   └── 加载 identity.md / perception.md
    │
    └── DesireActionExecutor 初始化
        └── 传入 ConsciousLiving 引用
```

## 2. 主循环流程（每秒）

```
ConsciousLiving._loop_awake()
    │
    ├── _tick_flame() ─────── 每秒
    │   │
    │   ├── Consciousness.tick_L0(agent_state)
    │   │   │
    │   │   ├── _sense() 收集感知数据
    │   │   │   ├── agent_state（运行状态）
    │   │   │   ├── user_idle_duration（用户空闲）
    │   │   │   ├── memory_count（记忆数量）
    │   │   │   └── recent_memory_summaries（最近记忆）
    │   │   │
    │   │   ├── Drive.tick()
    │   │   │   ├── tick_minute() → 情绪衰减
    │   │   │   └── tick_hour() → 激素/欲望自然变化
    │   │   │
    │   │   ├── SelfImage.update_from_perception()
    │   │   │   ├── 更新 SelfState（从 Drive 获取心情/能量）
    │   │   │   ├── 更新 SelfGrowth（从 Purpose 获取目标/进度）
    │   │   │   └── 更新 SelfMemory（最近记忆摘要）
    │   │   │
    │   │   └── 返回 FlameState
    │   │
    │   └── Consciousness.tick_L1() ─── 每60秒
    │       ├── 检测异常（用户长时间不活跃等）
    │       └── 返回 ConsciousnessReport
    │
    ├── _check_inner_thought() ── 每5分钟
    │   └── 生成内在想法（可选）
    │
    └── _check_dynamic_fuel() ──── 动态加柴判断
        │
        ├── 事件提取周期（10分钟）
        │   └── _extract_drive_events()
        │       ├── 获取最近对话
        │       ├── EventExtractor.extract()（LLM分析）
        │       ├── 提取表扬/批评/目标进展
        │       ├── Drive.on_praise() / on_criticism()
        │       ├── Drive.update_desire_from_llm()
        │       └── _check_desire_actions() ← 检查欲望行为
        │
        ├── 用户空闲 > 5分钟
        │   ├── _fuel_L2("user_idle_long")
        │   └── _check_desire_actions() ← 检查欲望行为
        │
        └── 定期加柴 > 10分钟
            └── _fuel_L2("periodic")
```

## 3. LLM 加柴流程（L2/L3）

```
Consciousness.tick_L2(context) 或 tick_L3()
    │
    ├── _build_deep_prompt()
    │   │
    │   ├── Self 部分
    │   │   ├── identity（我是谁）
    │   │   ├── role（当前角色）
    │   │   ├── environment（我在哪）
    │   │   └── consciousness_age（燃烧时长）
    │   │
    │   ├── Drive 部分 ←── Drive.get_state_text()
    │   │   ├── 当前心情（emotion.type + intensity）
    │   │   ├── 动力水平（motivation_level）
    │   │   ├── 压力水平（cortisol）
    │   │   └── 满足感（serotonin）
    │   │
    │   ├── Purpose 部分 ←── Purpose.get_state_summary()
    │   │   ├── 存在意义（meaning.identity）
    │   │   ├── 当前目标（current_goal.description）
    │   │   ├── 目标进度（goal_progress）
    │   │   └── 待执行目标数量
    │   │
    │   └── Memory 部分
    │       ├── memory_count
    │       └── recent_memory_summaries
    │
    ├── LLM 调用
    │   └── 生成意识报告 / 内在想法
    │
    └── 更新状态
        ├── SelfGrowth.inner_thought = 报告内容
        ├── SelfGrowth.inner_thought_history.append()
        └── clear_accumulated_changes()
```

## 4. 欲望驱动行为流程

```
_check_desire_actions()
    │
    ├── Drive.check_desire_actions()
    │   ├── 归属欲 > 0.7 → greet_user
    │   ├── 认知欲 > 0.8 → learn_topic
    │   ├── 成就欲 > 0.6 → progress_goal
    │   └── 表达欲 > 0.7 → express_idea
    │   └── 返回候选行为列表（按优先级排序）
    │
    └── DesireActionExecutor.execute_best_action()
        │
        ├── greet_user（归属欲）
        │   ├── 检查用户空闲 > 1分钟
        │   ├── LLM 生成问候消息
        │   ├── on_proactive(message) 发送
        │   └── Drive.on_desire_satisfied("belonging", 0.2)
        │
        ├── learn_topic（认知欲）
        │   ├── _get_learning_topic()
        │   │   ├── Purpose.get_current() → 当前目标描述
        │   │   └── identity.md learning_interests → 后备
        │   ├── LLM 搜索/生成知识
        │   ├── _save_knowledge(topic, content)
        │   │   └── ~/.xiaomei-brain/agents/{id}/knowledge/{topic}.md
        │   └── Drive.on_desire_satisfied("cognition", 0.3)
        │
        ├── progress_goal（成就欲）
        │   ├── Purpose.get_current() → 获取当前目标
        │   ├── 记录内在想法（推进意图）
        │   └──（后续实际推进目标）
        │
        └── express_idea（表达欲）
            ├── 检查内在想法存在
            ├── on_proactive(想法内容)
            └── Drive.on_desire_satisfied("expression", 0.2)
```

## 5. 目标完成反馈流程

```
Purpose.complete_current(success=True)
    │
    ├── Goal.complete()
    │   ├── status = COMPLETED
    │   └── progress = 1.0
    │
    ├── Drive.on_goal_completed()
    │   ├── RPE 计算（实际奖励 - 预期奖励）
    │   ├── Emotion → JOY（比预期好）
    │   ├── Hormone → dopamine↑, serotonin↑, cortisol↓
    │   ├── Motivation → 更新预期奖励（学习）
    │   └── Desire → achievement↓（满足）
    │
    └── 自动切换到下一个目标
        └── Purpose.get_next() → 最高优先级待执行目标
        └── Purpose.set_current(next_goal.id)
```

## 6. 苏醒流程

```
ConsciousLiving._on_wake()
    │
    ├── 初始化时间戳
    │   ├── _last_active = now
    │   ├── _last_fuel_time = now
    │   └── _last_event_extract_time = now
    │
    ├── 火焰点燃
    │   └── SelfImage.last_user_activity_time = now
    │
    └── 欲望状态检查 ←── 新增
        │
        ├── Drive.check_desire_actions()
        │   └── 返回候选行为
        │
        └── 按类型执行
            ├── learn_topic → 立即执行（不依赖用户）
            ├── progress_goal → 立即执行
            ├── greet_user → 等待时机（用户空闲时）
            └── 无触发 → 静静等待
```

## 8. 关键集成点

| 集成点 | 代码位置 | 数据流向 |
|--------|----------|----------|
| Consciousness 引用 Drive/Purpose | `core.py:__init__` | Drive/Purpose 作为参数传入 |
| tick_L0 调用 Drive.tick() | `core.py:tick_L0` | Drive 情绪衰减 |
| _build_deep_prompt 整合状态 | `core.py:_build_deep_prompt` | Drive/Purpose 状态注入 LLM |
| 学习主题从 Purpose 获取 | `action_executor.py:_get_learning_topic` | Purpose 当前目标优先 |
| 目标完成通知 Drive | `purpose_engine.py:complete_current` | Drive 激素变化 |
| ConsciousLiving 初始化三者 | `conscious_living.py:__init__` | Drive → Purpose → Consciousness |

## 11. 设计决定

| 项目 | 决定 |
|------|------|
| Drive 不需要 LLM 做数值判断 | 衰减用算法规则 |
| Purpose 目标分解 | LLM 自动分解 + 规则后备 |
| 学习主题优先级 | Purpose 当前目标 > identity.md 配置 |
| 目标完成反馈 | Purpose → Drive（激素变化） |
| 状态整合给 LLM | get_state_summary() 合并 Drive + Purpose + Memory |
| Consciousness 是整合层 | 只读取，不拥有 Drive/Purpose 数据 |
