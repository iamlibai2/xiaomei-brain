### 交互示例

```
You: 你好

You: 帮我写一个快速排序
小美: 当然可以！这是快速排序的 Python 实现：

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

## 核心架构

### 启动流程

```
start_all.py
├── Config.from_json()          # 加载 ~/.xiaomei-brain/config.json
├── AgentManager                # 管理 agents
│   ├── xiaomei/ (独立内存)
│   └── xiaoming/ (独立内存)
├── AgentGateway                # 消息路由
│   ├── FeishuChannel (xiaomei)
│   └── FeishuChannel (default)
└── WebSocket Server (8765端口)
```

### 消息路由逻辑

```python
# 路由优先级：
1. WebSocket 指定 agent_id（最高）
2. Bindings 配置：channel + account_id → agent_id
3. 默认 agent（最低）
```

### 线程模型

```
主进程 (asyncio)
├── _process_messages 协程 (处理消息队列)
└── WebSocket 线程 (lark-oapi 独立事件循环)
    └── queue.Queue (跨线程消息传递)
```

## 配置说明

### config.json 结构

```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "accounts": {
        "default": {"appId": "...", "appSecret": "..."},
        "xiaomei": {"appId": "...", "appSecret": "..."}
      }
    }
  },
  "bindings": [
    {
      "agentId": "xiaomei",
      "match": {
        "channel": "feishu",
        "accountId": "xiaomei"
      }
    }
  ],
  "agents": {
    "list": [
      {
        "id": "xiaomei",
        "name": "小美",
        "talent": "你叫小美，是一个温柔体贴的AI伴侣...",
        "workspace": "/home/iamlibai/.xiaomei-brain/agents/xiaomei"
      }
    ]
  },
  "xiaomei_brain": {
    "agent": {
      "max_steps": 10,
      "system_prompt": "你叫小美..."
    },
    "memory": {
      "dir": "",
      "similarity_threshold": 0.3,
      "max_topics": 100
    }
  }
}
```

### 重要目录

| 路径 | 说明 |
|------|------|
| ~/.xiaomei-brain/config.json | 主配置文件 |
| ~/.xiaomei-brain/agents/{id}/ | Agent 目录 |
| ├── talent.md | 系统提示词 |
| ├── memory/ | 记忆存储 |
| └── sessions/ | 会话历史 |
| ~/.xiaomei-brain/agents/default/ | 默认 Agent |
| ~/.xiaomei-brain/agents/xiaomei/ | 小美 Agent |
| ~/.xiaomei-brain/agents/xiaoming/ | 小明 Agent |

### 配置热更新

- 修改 `~/.xiaomei-brain/agents/{id}/talent.md` 即时生效
- 无需重启服务
- 其他配置变更需重启

## 常见问题

### 1. WebSocket 连接失败

```bash
# 检查端口
netstat -tuln | grep 8765

# 查看日志
tail -f /home/iamlibai/.xiaomei-brain/agents/default/memory/conversations/...
```

### 2. Agent 响应慢

- 正常：第一次启动需要加载 embedding 模型
- 后续响应速度取决于 LLM API

### 3. 飞书消息无响应

- 检查 Feishu App ID/Secret 是否正确
- 确认飞书群聊已开启机器人权限
- 查看服务日志是否有 WebSocket 连接错误

### 4. 切换 Agent 无效

- 确认 agent_id 拼写正确
- 检查 bindings 配置是否匹配
- 查看 Agent 是否已创建 `talent.md` 文件
